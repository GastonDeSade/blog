Pour trouver tous les fichiers contenant "blog-placeholder-1.jpg" dans votre dépôt local, vous pouvez utiliser la commande grep dans le terminal :
```bash
grep -r "blog-placeholder-1.jpg" .
```

Cette commande va:

-r : chercher récursivement dans tous les sous-répertoires
. : à partir du répertoire courant
chercher la chaîne exacte "blog-placeholder-1.jpg"
